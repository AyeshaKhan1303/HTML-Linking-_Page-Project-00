# Email Links Project

This project contains an HTML page with multiple useful links for contacting or viewing profiles of the author. It includes links to email, LinkedIn, GitHub, and a Vercel project. Additionally, there's a button linking to another HTML file (`Linking_to_a_specific_part_of_the_same_page.html`).

## Features

- **Contact via Email:**
  - A mailto link to quickly send an email to `seeetykhan30@gmail.com`.

- **LinkedIn Profile:**
  - Direct link to the author's [LinkedIn profile](https://www.linkedin.com/in/ayesha-khan-430124312/).

- **GitHub Profile:**
  - Direct link to the author's [GitHub profile](https://github.com/AyeshaKhan1303).

- **Vercel Projects:**
  - A link to view the author's projects hosted on [Vercel](https://vercel.com/ayesha-khans-projects-3ae3f43d).

- **Navigation Button:**
  - A button to navigate to the homepage (`Linking_to_a_specific_part_of_the_same_page.html`).

## How to Use

1. Open the `email_links.html` file in a browser.
2. Click on the desired link to be redirected to the corresponding platform or to send an email.
3. Use the button to navigate to the additional page as specified.
